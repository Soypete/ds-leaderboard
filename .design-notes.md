# Design notes — Dragonslayer leaderboards

## Brief (pinned)
Subject: leaderboards for a terminal RPG that teaches vim (bugs=dragons, reliable
code=gold). Audience: terminal-dwelling devs. Page job: show a player their rank +
make "how to get on the board" obvious.

## Thesis
A herald's standings board rendered as an **illuminated terminal ledger**. Keep the
dark Gruvbox terminal substrate (true to a CLI game, honors the existing palette),
but treat the board as a *ruled ledger*: champion pulled into a banner, ranks in a
numeric display face, gold as a tabular tally column.

Type (3 roles — currently everything is one mono, that's the miss):
- DISPLAY: "Cinzel" — a Roman inscriptional serif. Used ONLY for h1/champion handle
  + section eyebrows. Carries the "herald / engraved banner" personality. Restraint:
  never in body.
- BODY/UI: keep system mono stack (ui-monospace…) — it IS the subject (a CLI game).
- NUMERIC: "Fragment Mono" not available offline → use the mono stack with
  font-variant-numeric: tabular-nums + slashed-zero for ranks/gold so columns align
  like a ledger. Ranks get a large weight.

Layout: ledger with a champion banner on top.

## Signature
The **champion banner**: rank-1 lifted out of the table into an engraved gold-edged
plaque, handle in Cinzel, gold figure oversized in tabular numerals. One bold place;
the rest of the field stays quiet ruled rows.

## Numbering note
Rank numerals ARE real sequence info (it's a leaderboard) — so numbered markers are
justified here, not decoration. Use circled/engraved numerals ❶❷❸ for the podium feel.

## Critique log
- (pass 1) avoid default #2 (black + 1 acid accent): we keep dark but add the ledger
  rules + 2-face type + champion banner so it's not "dark with one accent + a table".
- Cinzel is a risk (could read wedding-invite). Justify: it's inscriptional/engraved,
  matches "herald's standing banner", and is used ONLY for the champion + h1, against
  a terminal mono field — the contrast (engraved serif vs CLI mono) is the point.
- (pass 2 / "remove one accessory") CUT the Roman-numeral Ⅰ (U+2160) from the champion
  banner — it's tofu-box-prone and it was THE signature element, so a fallback box there
  was unacceptable. Replaced with a plain ASCII "1" in Cinzel + a gold rule + a small
  crown (♔, decorative ::before, aria-hidden → safe to degrade). Lesson: never put an
  exotic codepoint in the one element the whole page rests on.
- Gotcha: running `npm run build` while `next dev` is live writes a prod .next that
  collides with dev chunks → "Cannot find module './NNN.js'" 500s. Fix: rm -rf .next +
  restart dev. Run the prod build only as a FINAL check, not mid-iteration.
